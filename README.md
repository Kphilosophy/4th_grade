# 研究のための統計手法入門

高校生向けに、研究活動で使えるPythonを使用するためのアイデア集です。

## 対象者

- 課題研究・探究活動でデータ分析が必要な学生
- 統計の基礎(平均・分散)を学習済みであること

## 内容

| No. | テーマ | ノートブック |
|-----|--------|--------------|
| 01 | t検定 | notebooks/01_t-test.ipynb |
| 02 | 分散分析 | notebooks/02_anova.ipynb |
| 03 | 回帰分析 | notebooks/03_regression.ipynb |

## 環境構築

```bash
git clone https://github.com/ユーザー名/リポジトリ名.git
cd リポジトリ名
pip install -r requirements.txt
```

Google Colab で開く場合はインストール不要です。
各ノートブック冒頭のColabバッジをクリックしてください。
